# 🚀 CONVERTIDOR PRO (Limpio) ✨

¡Bienvenido/a a este increíble y útil proyecto! 🎉 Este es un práctico script de Python 🐍 diseñado para hacer conversiones numéricas entre diferentes bases matemáticas. 💡

## 🌟 ¿Qué hace esta maravilla?
Este programa te permite convertir números de forma rápida ⚡ y sencilla. ¡Solo eliges una opción en el menú interactivo y obtienes resultados instantáneos! ⏱️

### 🛠️ Funciones Principales
- **Decimal a Binario, Octal y Hexadecimal** 🔟
- **Binario a Decimal, Octal y Hexadecimal** 👽
- **Hexadecimal a Decimal, Binario y Octal** 🔠
- **Octal a Decimal, Binario y Hexadecimal** 🎱
- **Limpieza automática de pantalla** en Windows 🪟 y Linux/Mac 🐧 ¡Para mantener todo ordenado! ✨
- **Protección contra errores** (si escribes algo mal, el programa te avisará amablemente y no fallará 🛡️).

## 🎮 ¿Cómo usarlo?
1. Descarga el archivo `convertidor.py` en tu computadora. 💻
2. Abre tu terminal o línea de comandos. ⬛
3. Ejecuta el comando: `python convertidor.py` 🚀
4. ¡Sigue las instrucciones en pantalla y diviértete calculando! 🔢

## 🤖 Requisitos
- ¡Solo necesitas tener Python instalado en tu sistema! 🐍 (Cualquier versión 3.x funcionará).
